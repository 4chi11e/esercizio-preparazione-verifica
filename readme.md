# HTML - Esercizio di preparazione per la verifica

Questo piccolo progetto è stato creato come esempio pratico contenente tutti gli elementi che possono essere richiesti in una verifica di un'ora a scuola. 

Questo progetto è chiaramente troppo lungo per essere svolto in un'ora, nella verifica saranno chieste meno cose ma simili a quelle qui sviluppate.

## Versione menu a tendina

In questa versione del progetto è stato aggiunto un menu a tendina che normalmente non viene chiesto in verifica perchè lungo e complesso per una verifica di una sola ora.
